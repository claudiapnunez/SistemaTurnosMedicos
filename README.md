# SistemaTurnosMedicos - Diseño Orientado a Objetos

- **Materia:** Diseño Orientado a Objetos.
- **Carrera:** Tecnicatura Universitaria en Programación de Sistemas.
- **Grupo:** Grupo N° 1.

## Introducción

Este proyecto tiene el objetivo de desarrollar el diseño orientado a objetos de un sistema de gestión de turnos para un consultorio médico, aplicando tarjetas CRC, diagramas de casos de uso y escenarios de casos de uso.
